Want to store different passwords for everything

all inside your own brain?

No problemo friend! Here's how...

Step 1: THE BASE

Start with a base word for your password. Now, we want this word to be different for each site you use, so let's use the website name as the base - easy!

Example password base: google

Step 2: THE ENCRYPTION

Now this part might sound intimidating, but it doesn't need to be! Start with something easy - like adding an exclamation mark at the end of the password perhaps?

Example password encryption: google!

Step 3: THE KEY

Now you have your base and encryption, you've done it! You have a password that is unique to every website. All you have to do is remember your encryption key so you can apply it to every site.

Example password key: Add an exclamation mark at the end of the password

Sounds simple enough right? Only catch is that this example is a bit too easy to guess, so you need to add a few more encryptions to make the password more secure. Want to give it a try for yourself?
