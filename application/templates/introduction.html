Want to store different passwords for everything? All inside your own brain?

No problemo friend!

Yes, you really do.

As you know, our online accounts contain information that we don't want random people on the internet to get hold of. Automated machines now exist in the world that are built to gain unauthorised access to our online stuff. It's sad and it's true.

An evil machine like this will keep trying out different passwords on an account until it guesses correctly. Terrifyingly, these machines can try thousands of character combinations per second. Yes, thousands! Per second!

Once these machines have cracked one of our passwords, they can get into all of our other accounts with the same password. And that is why we really, really, really need to have different passwords for every single one of our accounts.

The more complex our passwords are, the more security they are going provide for our online stuff. We like security, security is great. So let's make our passwords more complex! We can use encyption :)

Encryption is a process to convert text into secret code. The only way to read that secret code is with the key!

An example of an encryption key could be:

'replace each letter of the text with the next letter in the alphabet'

With this key, 'A' would become 'B', 'B' would become 'C' and so on. The secret code for the text 'HELLO' would be 'IFMMP'.

Anyone with the key would be able to decifer any secret code that has been encrypted using that key.

Make sense? If not, have a quick google!

So here on this site, we are going to use encryption to create passwords for all of your accounts! Sounds intimidating, but I've made it easy for you buddy.

The best part is... you only have to create one key that will encrypt all of your passwords. One key (you might say) to rule them all.

The text that we will encrypt to make your password - is the name of the site that you are creating an account for.

For example, lets say your secret key is:

add a '#' to the end of the text

Your password for facebook would be 'facebook#' or your password for instagram would be 'instagram#' and so on...

Now, that example key is just a little bit too easy to guess, so you'll want to make yours more elaborate.

Once you are happy with the security of your key, as long as you have it memorised, you can remember all of your passwords!

Doesn't that sound wonderful? Come on pal, you got this!
